
# Rana Export Next.js Project

## Installation

1. **Clone the repository:**
	```bash
	git clone <repository-url>
	cd rana_export
	```

2. **Install dependencies:**
	```bash
	npm install
	```

3. **Run the development server:**
	```bash
	npm run dev
	```

4. **Open your browser:**
	Visit [http://localhost:3000](http://localhost:3000) to view the app.
